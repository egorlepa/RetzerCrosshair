# RetzerCrosshair

A lightweight cursor crosshair overlay for World of Warcraft. Draws a ring and four lines extending to the screen edges, centered on your mouse cursor at all times.

## Features

- Ring around the cursor with configurable radius
- Full-screen crosshair lines extending from the ring to each screen edge
- Combat-only or always-on mode
- Hides automatically during mouselook (right-click camera)
- Fully configurable color with alpha, line width, and ring size

## Usage

Type `/rch` to open the settings panel. All options take effect immediately — no reload required.

| Setting          | Default       | Description                              |
|------------------|---------------|------------------------------------------|
| Color            | Red, 85% alpha | Color and opacity of all lines          |
| Ring Radius      | 20px          | Radius of the ring around the cursor     |
| Line Width       | 4px           | Thickness of all lines                   |
| Show             | Combat only   | Show in combat only, or always           |
| Hide on Mouselook | On           | Hide while right-click camera is active  |
